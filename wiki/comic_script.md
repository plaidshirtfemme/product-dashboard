# Комикс Motif — сценарий A1 «Потеря арта» (DASH-100)

> Финальный сценарий + раскадровка (сцена-уровень). Источник истории — `team_world_bible.md`.
> Проекция на дашборд — раздел «Сценарий» вкладки About (`motif_world.py → A1_SCRIPT_FINAL`).
> Формат комикса: **СТРАНИЦЫ**. Отрисовка — AI (DASH-101, выбор инструмента — критерий: консистентность
> персонажа между кадрами). Дальше по пайплайну — **кадр-уровень** (панели) + character sheets.

## Принципы подачи
1. **Доступно не-IT:** каждый техтермин → сноска простыми словами (миграция, деплой, техдолг, QA, retention).
2. **Грунтинг раскидан на ВСЕХ** (из world-bible §2N), не только на одного героя.
3. **Реакции — по характерам**, не хором (см. развилку ценностей).
4. **Ансамбль:** никто не выпячен; PD-вклад (Ая) виден наравне.
5. **Анти-приторность:** не «дружная семья»; человечность обоснована бизнесом (риск поставки, удержание таланта).

## Механика кризиса (компетентная — переживёт въедливого читателя)
Марко — сильный мидл, не халатность. Потеря данных = связка:
- **Живую миграцию** гнали без полной заморозки записи (избегали даунтайма перед демо — трейд-офф под срок).
- **Латентный недокументированный техдолг** старого хранилища (bus-factor=1 бэкенд, Марко один, некому ревьюить).
- Одновременные записи юзеров + техдолг → часть активно редактируемых проектов побита.
- **Инфра-бэкапы (ночные) БЫЛИ** → вернули старое, но **дельту после последнего бэкапа** (свежую работу, дни Нура)
  не спасти — **пользовательскую version-history задвинули** (RPO-разрыв).
- Причинно: латентный техдолг + живая миграция + RPO — **НЕ «забыли бэкапы»**. Ответственность Motif (единый источник правды).
- Починка ~2 дня, дельта безвозвратна → честное извинение + компенсация. Фикс: непрерывная version-history.

## Сценарий по дням (спайн: Ставка·0 / Нарастание·1–8 / Кризис·9–10 / Развязка·11 / Послевкусие·12–13)

**Фон весь спринт:** короткий runway · найм заморожен · все в нескольких ролях · сообщество тихо растёт (калм).

- **День 0 · Ставка.** Дэн запускает спринт (v2 к демо, ставка = удержание → Series A). Ким переобещает.
  Марко: под v2 нужна миграция; чтобы без даунтайма — вживую; version-history задвигают. Предвестия: техдолг, «сообщество ждёт обещанного».
- **День 1.** ✨ Идеальный кандидат-БЭКЕНДЕР из ниоткуда — найм заморожен. Марко (bus-factor=1) придерживает резюме.
- **Дни 2–4.** У Дэна переезд/протечка (до дня 11). ✨ Юзер-вирал (4–5): Ли раздувает, Прия видит рост.
- **День 5.** Латентный техдолг «дозрел» — тихо.
- **Дни 6–8.** Слабый QA: Том сигналит, Лея (СДВГ) спешит-катит. День 7 — старт тихого выгорания Леи (маска).
- **День 9 · КРИЗИС.** Живая миграция + записи → побита свежая работа; дельту не спасти. Нур над пустым холстом. Ли — вал гнева. Ким хочет замять.
- **Ночь дня 9.** Заночевали: Марко чинит; Ая (recovery-UX рядом), Лея (вина-помощь), Ли (кофе/настроение), Том заглядывает.
- **День 10 · Развязка.** ~2 дня починки. Реми — blameless-постмортем: системное (bus-factor, техдолг, живая
  миграция, придержанный кандидат, выгорание Леи — её видят). Ая — recovery-UX + извинение + компенсация. Софи+Прия: доверие=удержание. ☀️ погода.
- **День 11.** Демо Даниэлю честно про инцидент И реакцию → доверие растёт.
- **Дни 12–13 · Послевкусие.** 🏆 Нур — награда за комикс. Series A → размораживают найм → бэкендера берут.
  Непрерывная version-history как принцип. Storming → norming.

## Раскадровка (сцена-уровень) — 12 сцен
0. **Пролог — что такое Motif** (разбросанная команда / хаос PSD ↔ общий холст).
1. **Ставка** (стендап с кофе; ставка спринта).
2. **Решают ускорить** (миграция нужна под v2; вживую; Том предупреждает — перекрыли).
3. **Ирония кандидата** (бэкендер, найм заморожен; Марко тихо закрывает).
4. **Нарастание** (монтаж: каждый за своей ролью + грунтинг у каждого + вирал).
5. **Тихая тень** (техдолг дозрел; Лея маскирует выгорание).
6. **КРИЗИС** (потеря свежей работы; Нур над пустым холстом; гнев; Ким замять).
7. **Развилка ценностей** (реакции по характерам → честность).
8. **Ночь — заночевали** (Марко + Ая/Лея/Ли/Том; тепло, не приторно).
9. **Постмортем** (Реми фасилитирует; системное вскрыто; каждый за ролью).
10. **Демо Даниэлю** (честность → доверие).
11. **Послевкусие** (награда Нура; Series A; найм; version-history).
12. **Эпилог** (почему остаются: приняли, работу видят, среда работает — анти-«семья»).

## Роли-активности в кадре (сверка с USER_STORIES)
Каждый герой делает свою настоящую работу: Дэн — планинг/координация; Марко — ADR/миграция/incident;
Прия — воронки/retention; Ая — дизайн v2 / recovery-UX / UX-writing; Лея — сборка фронта (сильный выхлоп);
Том — QA-предупреждение/верификация; Ким — видение/демо/решение; Ли — сообщество/тикеты; Реми — фасилитация;
Софи — ресёрч (доверие=удержание); Нур — dogfood-юзер/крафт. Обогатить: Софи (ресёрч-бит), Ая (v2-дизайн), Лея (сильный выхлоп).
